 2. Описание проекта

Проект направлен на сравнительный анализ библиотек Matplotlib, Seaborn и Plotly.
Изучение их функциональности и удобство использования.

Основные задачи:
- исследование основных характеристик и возможностей библиотек Matplotlib, Seaborn, Plotly.
- определить основные области применения каждой библиотеки.
- выбрать или подготовить набор данных для визуализации данных.
- реализовать визуализацию с помощью Matplotlib, Seaborn и Plotly.
- сравнить функциональные возможности использованных библиотек и удобство использования.


3. Установка проекта

Создать проект в удобной IDE.

Открыть и запустить необходимый файл.

    3.1 Клонируйте проект с GitHub:

    git clone https://github.com/DanilTer03/-.git


    3.2. Перейдите в папку проекта:

    cd ваш_проект

    3.3 Создайте виртуальное окружение (если используется):

    python -m venv venv
    source venv/bin/activate  # Для macOS/Linux
    venv\Scripts\activate  # Для Windows


    3.4 Установите зависимости:

    Установить пакеты библиотек, с помощью команд в терминале среды разработки:
    pip install -r requirements.txt

4. Использование проекта

## Использование проекта

Запуск основного скрипта:

python main.py

6. Структура проекта

## Структура проекта

- main.py - Главный файл запуска
- matplotlb.py - Демонстрация простого линейного графика с помощью библиотеки matplotlib
- seabrn.py - Демонстрация простого линейного графика с помощью библиотеки seaborn
- pltly.py - Демонстрация простого линейного графика с помощью библиотеки plotly
- matplotlb3d.py - Демонстрация 3D визуализации двойной спирали с помощью библиотеки matplotlib
- seabrn3d.py - Демонстрация 3D визуализации двойной спирали с помощью библиотеки seaborn
- pltly3d.py - Демонстрация 3D визуализации двойной спирали с помощью библиотеки plotly
- README.md - Документация проекта
- requirements.txt - Список зависимостей
